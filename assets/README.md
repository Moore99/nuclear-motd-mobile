# Nuclear MOTD Mobile Assets

Place your app assets in these folders:

## images/
- logo.png - App logo
- splash.png - Splash screen background
- placeholder.png - Placeholder for images

## icons/
- app_icon.png - App icon (1024x1024 for generation)
- notification_icon.png - Push notification icon

## fonts/
Download Roboto fonts from Google Fonts and place here:
- Roboto-Regular.ttf
- Roboto-Medium.ttf
- Roboto-Bold.ttf
