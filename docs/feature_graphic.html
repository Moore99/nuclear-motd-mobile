Instructions to Generate Feature Graphic:

  1. Open this HTML file in your browser (Chrome recommended)
  2. Right-click on the blue graphic above
  3. Select "Inspect" or "Inspect Element"
  4. In DevTools, right-click on the <div class="feature-graphic"> element
  5. Choose "Capture node screenshot" (Chrome) or use browser extension to screenshot the exact element
  6. Alternative: Take a screenshot and crop to exactly 1024x500 pixels
  7. Save as PNG and upload to Google Play Console

Or use online tool: Open this file and use a screenshot tool like Nimbus Screenshot or GoFullPage to capture just the blue graphic area.
